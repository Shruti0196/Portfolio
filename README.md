# Portfolio Website - Shruti Prasad

A modern, aesthetic portfolio website with unique interactive features built with HTML, CSS, and JavaScript.

## ✨ Features

### Unique Interactive Features
- **Particle Animation Background** - Dynamic particle system with connecting lines in the hero section
- **Typing Animation** - Animated name typing effect with blinking cursor
- **Dark/Light Mode Toggle** - Smooth theme switching with localStorage persistence
- **Glassmorphism Effects** - Modern frosted glass design elements
- **Smooth Scroll Animations** - Fade-in animations triggered on scroll
- **Animated Skill Bars** - Progress bars that animate when scrolled into view
- **Interactive Experience Tabs** - Tabbed interface for work experience
- **3D Hover Effects** - Transform and shadow effects on cards and buttons
- **Responsive Design** - Fully responsive across all device sizes
- **Active Navigation Highlighting** - Dynamic navigation link highlighting on scroll

### Sections
1. **Hero Section** - Eye-catching introduction with particle background
2. **About** - Personal introduction and technologies
3. **Experience** - Interactive tabbed work experience
4. **Projects** - Featured projects with hover effects
5. **Skills** - Animated skill bars and technology tags
6. **Education** - Timeline-style education section
7. **Contact** - Contact information and social links

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! The portfolio is ready to use.

### Local Development

For a better development experience, you can use a local server:

**Using Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Using Node.js:**
```bash
npx serve
```

Then open `http://localhost:8000` in your browser.

## 🎨 Customization

### Colors & Theme
- Edit the CSS variables in `styles.css` under `:root` and `[data-theme="light"]`
- Main colors:
  - `--bg-primary`: Primary background color
  - `--bg-secondary`: Secondary background color
  - `--accent`: Accent color (default: #64ffda)
  - `--text-primary`: Primary text color
  - `--text-secondary`: Secondary text color

### Content
- Update personal information in `index.html`
- Modify section content directly in the HTML
- Update project links and descriptions
- Add/remove skills and technologies

### Animations
- Adjust animation durations in CSS `--transition` variable
- Modify particle count in `script.js` (particleCount variable)
- Customize typing speed in `typeEffect()` function
